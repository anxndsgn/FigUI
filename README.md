# FigUI

FigUI is a Figma UI3 component library for building Figma plugin user interfaces with a familiar, consistent look and feel.

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm preview
```

## Deploy (Cloudflare Workers)

```bash
pnpm deploy
```

## Registry

Regenerate installable JSON files:

```bash
pnpm registry:build
```
