## Installation

Install the input package, which includes `ColorInput`:

<CodeSnippet>
  <CodeSnippet.Command value='pnpm'>
    pnpm dlx shadcn@latest add http://localhost:3000/r/input.json
  </CodeSnippet.Command>
  <CodeSnippet.Command value='npm'>
    npx shadcn@latest add http://localhost:3000/r/input.json
  </CodeSnippet.Command>
  <CodeSnippet.Command value='bun'>
    bunx shadcn@latest add http://localhost:3000/r/input.json
  </CodeSnippet.Command>
</CodeSnippet>

Also install `chroma-js`:

```bash
pnpm add chroma-js
```

## Usage

```tsx
import { ColorInput } from '@/components/ui/color-input';

<ColorInput value={color} onValueChange={setColor} />
```

Values are normalized to uppercase hex without a leading `#` when committed (blur or Enter).

## Props

| Prop | Type | Description |
| ---- | ---- | ----------- |
| `onValueChange` | `(next: string) => void` | Called with the normalized hex string (uppercase, no leading `#`) when the user commits the value |
