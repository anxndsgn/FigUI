## Installation

Install the input package, which includes `TextInput`:

<CodeSnippet>
  <CodeSnippet.Command value="pnpm">
    pnpm dlx shadcn@latest add https://figui.dev/r/input.json
  </CodeSnippet.Command>
  <CodeSnippet.Command value="npm">
    npx shadcn@latest add https://figui.dev/r/input.json
  </CodeSnippet.Command>
  <CodeSnippet.Command value="bun">
    bunx shadcn@latest add https://figui.dev/r/input.json
  </CodeSnippet.Command>
</CodeSnippet>

## Usage

```tsx
import { TextInput } from "@/components/ui/text-input";

<TextInput placeholder="Search..." />;
```

When placed inside an `InputGroup`, the input automatically strips its individual ring, background, and rounding to inherit the group shell.
