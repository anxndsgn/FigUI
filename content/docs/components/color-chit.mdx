A compact color swatch for input groups. The left half shows the solid color; the right half previews opacity against a checkerboard background.

## Usage

`color` accepts a hex string without a leading `#`. Pass `opacity` as a 0–100 percentage to split the preview between solid and translucent halves.

```tsx
<ColorChit color="FF24BD" />
<ColorChit color="FF24BD" opacity={72} />
```

Inside an input group, place it in `InputGroupAddon` beside a `ColorInput`:

```tsx
<InputGroup>
  <InputGroupAddon>
    <ColorChit color={color} opacity={opacity} />
  </InputGroupAddon>
  <ColorInput value={color} onValueChange={setColor} />
</InputGroup>
```

## Props

| Prop | Type | Description |
| ---- | ---- | ----------- |
| `color` | `string` | Hex color without `#` (3, 6, or 8 digits) |
| `opacity` | `string \| number` | Optional opacity percentage (0–100) for the right half |
| `className` | `string` | Additional classes for the root element |
| `onClick` | `MouseEventHandler` | Optional click handler; adds `cursor-pointer` when set |
