## Installation

<CodeSnippet>
  <CodeSnippet.Command value='pnpm'>
    pnpm dlx shadcn@latest add http://localhost:3000/r/slider.json
  </CodeSnippet.Command>
  <CodeSnippet.Command value='npm'>
    npx shadcn@latest add http://localhost:3000/r/slider.json
  </CodeSnippet.Command>
  <CodeSnippet.Command value='bun'>
    bunx shadcn@latest add http://localhost:3000/r/slider.json
  </CodeSnippet.Command>
</CodeSnippet>

Install `chroma-js` for the color range and opacity sliders:

```bash
pnpm add chroma-js
```

## Usage

FigUI provides three slider variants:

- `Slider` — default numeric slider
- `ColorRangeSlider` — hue picker (0–360)
- `OpacitySlider` — opacity picker (0–1) with a color preview
