## Usage

```tsx
import { Button } from '@/components/ui/button';

<Button>Button</Button>
<Button variant='secondary' size='large'>Large</Button>
```

Use the `render` prop to render a link button:

```tsx
<Button render={<a href='/'>Home</a>} />
```

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `variant` | `'primary' \| 'secondary' \| 'destructive' \| 'secondaryDestruct' \| 'inverse' \| 'success' \| 'link' \| 'linkDanger' \| 'ghost'` | `'primary'` | Visual style variant |
| `size` | `'default' \| 'large' \| 'icon'` | `'default'` | Button size |
