## What is FigUI?

FigUI is a open source collection of Figma UI3 react components. You can use it in your figma plugin to create a seamless experience.

## Stack

- [react](https://react.dev/)
- [tailwindcss](https://tailwindcss.com/)
- [baseui](https://base-ui.com/)
- and specially the design system from [figma ui3 library](https://www.figma.com/community/file/1486123838948777078)

## How to use

Just like [shadcn/ui](https://ui.shadcn.com/), you can copy the code from the website and paste it in your code base.
