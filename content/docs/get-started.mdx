## Installation

First, you need to install [base-ui](https://base-ui.com/react/overview/quick-start) and [tailwindcss](https://tailwindcss.com/docs/installation).

```bash
pnpm install @base-ui/react
```

Second, add cn helper function following the shadcn/ui [documentation](https://ui.shadcn.com/docs/installation/manual).

```bash
pnpm add class-variance-authority clsx tailwind-merge lucide-react
```

```tsx
import { clsx, type ClassValue } from 'clsx';
import { twMerge } from 'tailwind-merge';

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

Third, add css to your project, you can paste the following code to your `globals.css` file or create a new one and import it in your `globals.css` file.

Note that if you are using figui in the figma plugin, you need to change the `@custom-variant dark (&:is(.dark *, [data-theme="dark"] *));` to `@custom-variant dark (&:is(.figma-dark *));` in your `globals.css` file for dark mode to work properly.

Fourth, set up portals so Base UI popups such as Select, Dialog, and Tooltip render above your page content. Wrap your application layout in a `.root` element and add the following styles to your global CSS. See the [Base UI quick start](https://base-ui.com/react/overview/quick-start) for details.

```tsx
<body>
  <div className="root">
    {children}
  </div>
</body>
```

```css
body {
  position: relative;
}

.root {
  isolation: isolate;
}
```

`isolation: isolate` keeps your app's `z-index` values from interfering with portaled popups. `position: relative` on `body` ensures dialog backdrops cover the full viewport on iOS 26+ Safari.
